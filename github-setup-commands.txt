GitHub Repository Setup Commands
=====================================

After creating a repository named 'aleppogift' on GitHub.com, run these commands:

1. Add GitHub as remote origin:
   git remote add origin https://github.com/mmurhaf/aleppogift.git

2. Push to GitHub:
   git branch -M main
   git push -u origin main

Alternative Commands if you named the repository differently:
=============================================================

If your repository has a different name, replace 'aleppogift' with your repo name:
   git remote add origin https://github.com/mmurhaf/YOUR-REPO-NAME.git
